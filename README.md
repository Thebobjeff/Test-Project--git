# LangChain Groq Document Summarizer

A Python-based tool that utilizes LangChain and Groq's Llama models to automatically summarize Markdown (`.md`) files.

## 🚀 Features
- **Fast Inference:** Powered by Groq's LPUs for near-instant summaries.
- **Smart Parsing:** Uses `unstructured` to handle Markdown formatting.
- **Secure:** Environment variable support via `.env` to protect API keys.

---

## 🛠️ Prerequisites

- **Python 3.10+** (You are using 3.13)
- **Groq API Key:** Get yours at [console.groq.com](https://console.groq.com/)
